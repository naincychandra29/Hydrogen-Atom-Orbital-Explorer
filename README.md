# Hydrogen Atom Orbital Explorer

This project visualizes the probability density of hydrogen atom orbitals using Python.

The program uses mathematical wavefunctions and spherical harmonics to generate and display a 2D slice of the electron probability distribution for different quantum numbers.

I made this project while exploring scientific computing, data visualization, and basic quantum physics concepts using Python.

## Features
- Hydrogen orbital visualization
- 3D grid computation
- Probability density calculation
- 2D slice plotting using Matplotlib
- Uses spherical harmonics and Laguerre polynomials

## Technologies Used
- Python
- NumPy
- Matplotlib
- SciPy

## How to Run
1. Install the required libraries: pip install numpy matplotlib scipy
2. Run the Python file

## Conepts Used
- Quantum numbers (n,l,m)

- Probability density visualization
